# Laika

Uma aplicação para monitoramento de arquivos de texto e persistência em um banco vetorial, permitindo queries que retornam valores relevantes para uma busca semântica.

Criado como uma forma de organizar minhas anotações acadêmicas em uma interface simples para busca e visualização por conteúdo indexado.
